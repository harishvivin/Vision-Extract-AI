1. EXECUTIVE PROJECT SUMMARY
--------------------------------------------------------------------------------
Vision Extract AI is an automated, production-grade Artificial Intelligence (AI) 
system designed for Document Intelligence, Zero-Shot Object Detection, and Pixel-
Accurate Instance Segmentation.

The project takes multi-page assignment PDF documents containing photographs 
and natural language questions, automatically extracts the images and questions, 
uses state-of-the-art Computer Vision & NLP models to identify the exact requested 
target objects, segment their precise outlines, and export cropped high-resolution 
PNG images along with execution telemetry logs.

Key Value Proposition:
- Fully Automated: Eliminates manual image cropping and object annotation.
- Zero-Shot AI Capability: Detects any object without needing specialized retraining.
- Pixel-Accurate Segmentation: Uses Meta AI's SAM 2 for sharp object masks.
- Multi-Platform Deployment: Cloud / Local FastAPI Backend + GitHub Pages Frontend.


================================================================================
2. CORE AI MODEL ARCHITECTURE & STACK
================================================================================

A. Computer Vision Stack:
   1. Grounding DINO (IDEA Research / Hugging Face):
      - Zero-shot vision-language object detection transformer.
      - Accepts open-vocabulary text prompts (e.g., "yellow tulips", "silver car", 
        "green balloon", "pink flamingo") and predicts bounding boxes.

   2. SAM 2 — Segment Anything Model 2 (Meta AI):
      - Next-generation promptable instance segmentation model.
      - Converts bounding boxes into pixel-accurate foreground object masks.

B. NLP & Document Intelligence Stack:
   1. PyMuPDF (fitz):
      - High-performance PDF renderer extracting 300 DPI page photographs and text streams.
   2. spaCy & Regex Entity Parser:
      - NLP parser extracting target object names, color attributes, spatial position 
        descriptors, and target output filenames from question text.

C. Full-Stack Web Architecture:
   1. Backend: FastAPI (Python REST API) with Uvicorn server.
   2. Frontend: React 18, Vite, Tailwind CSS, Lucide Icons.
   3. Hosting: GitHub Pages with client-side PDF rendering & Web APIs.


================================================================================
3. STEP-BY-STEP PIPELINE WORKFLOW (HOW IT WORKS)
================================================================================

[Step 1: Document Ingestion & Page Rendering]
- The user uploads a PDF file (e.g., INPUT_images_and_questions.pdf).
- PyMuPDF renders each PDF page at 300 DPI high resolution and extracts raw text.

[Step 2: NLP Question & Metadata Extraction]
- The NLP Question Parser parses sentence structure (e.g. "Crop out only the 
  SILVER MacBook Air (centre)..."):
  * Target Object: "MacBook Air" / "laptop"
  * Color Attribute: "SILVER"
  * Spatial Location: "centre"
  * Output Filename: "06_macbook_air.png"

[Step 3: Zero-Shot Object Detection & Spatial Position Scoring]
- Grounding DINO receives the photo and prompt ("silver MacBook Air").
- If multiple candidates exist, the Spatial Disambiguation Scorer calculates:
  Score = Model Confidence * (0.6 + 0.4 * SpatialMatch)
- Selecting the silver MacBook Air in the center while excluding other laptops.

[Step 4: SAM 2 Instance Masking]
- SAM 2 takes the bounding box prompt and generates a pixel-level segmentation mask.

[Step 5: Object Cropping & Output Packaging]
- The Image Cropper crops out the precise target object, saves the output PNG, 
  creates a green SAM 2 mask preview overlay, and packages all output PNGs into 
  "all_extracted_objects.zip".

4. REAL-WORLD CASE STUDY (EXAMPLE TO EXPLAIN TO HR)
================================================================================

Example Page: Image 06 (Laptops on a Desk)
Question Text: "Question 6: This image shows laptops on a desk. Crop out only the 
               SILVER MacBook Air (centre) and save the cropped region as 
               06_macbook_air.png."

How the Model Handles It:
1. Identifies 3 laptops on the desk (Dell left, MacBook Air center, Lenovo right).
2. NLP Parser extracts color "SILVER", object "MacBook Air", position "centre".
3. Spatial Disambiguation Scorer evaluates all 3 laptops:
   - Dell (Left) -> Rejected (Position mismatch)
   - Lenovo (Right) -> Rejected (Color & Position mismatch)
   - Silver MacBook Air (Center) -> MATCHED (94.1% Confidence score)
4. SAM 2 segments the exact aluminum body outline of the MacBook Air.
5. Saves clean crop as "06_macbook_air.png".
