1. EXECUTIVE SUMMARY & PROJECT OVERVIEW
----------------------------------------------------------------------------------------------------
Vision Extract AI was initially architected as a general-purpose OCR and PDF image extraction system. 
To meet strict medical underwriting standards and handle complex 20-page medical insurance reports 
(specifically for Tata AIA Life Insurance & Jeevandeep Diagnostic), the application was completely 
transformed into a specialized "Medical Report PDF Analyzer & Visual QA Engine".

Key Capabilities Added:
• 2-Stage User Workflow: PDF Upload & Medical Document Verification -> Unlocked Interactive Question Answering Bar.
• Multi-Modal Visual QA Engine: Trained on 100 Ground-Truth Medical Report Questions & Answers (Demographics, CBC, Biochemistry, HbA1c, Serology, LFT, Lipid Profile, Urine, ECG, Lifestyle, and Executive Summaries).
• Automatic Screenshot Evidence Retrieval: Renders high-res 200 DPI document page previews and dynamically crops normalized bounding boxes with emerald highlight borders for visual proof.
• Dual-Mode Execution Architecture: Seamless fallback between local Python/FastAPI ML server (/api/qa/ask) and high-speed client-side React evaluation engine for GitHub Pages static hosting.

4. BOUNDING BOX CROPPING & VISUAL EVIDENCE PIPELINE
----------------------------------------------------------------------------------------------------
When a user asks a question, the pipeline performs the following steps:
1. Query Intent Classification: Match query against trained dataset patterns.
2. Page & Bounding Box Isolation: Retrieve exact page number (1-20) and normalized coordinates [x1, y1, x2, y2].
3. High-Res Rendering: Open PDF page using PyMuPDF (fitz) at 200 DPI.
4. Dynamic Bounding Box Crop:
   crop_x1 = int(bbox[0] * Image_Width) - padding
   crop_y1 = int(bbox[1] * Image_Height) - padding
   crop_x2 = int(bbox[2] * Image_Width) + padding
   crop_y2 = int(bbox[3] * Image_Height) + padding
5. Outline Highlight: Draw 5px emerald border (#10b981) around the cropped snippet.
6. Evidence Rendering: Display crop preview in UI with 1-click modal expansion & PNG download.
